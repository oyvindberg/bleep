# Your First Bleep Project

This tutorial walks you through creating a simple Scala project with bleep from scratch.

## Prerequisites

- [Bleep installed](/docs/installing)
- A terminal
- A text editor or IDE

## Step 1: Create the Project Directory

```bash
mkdir my-first-bleep-project
cd my-first-bleep-project
```

## Step 2: Create bleep.yaml

Create a file named `bleep.yaml` with the following content:

```yaml
$schema: https://raw.githubusercontent.com/oyvindberg/bleep/master/schema.json
$version: 0.0.12

jvm:
  name: graalvm-community:21

projects:
  myapp:
    scala:
      version: 3.3.3
    platform:
      mainClass: myapp.Main
```

Let's break down what each part means:

- `$schema` - Enables IDE autocomplete and validation
- `$version` - The bleep version (should match your installed version)
- `jvm.name` - The JVM to use (bleep manages this for you)
- `projects.myapp` - Defines a project named "myapp"
- `scala.version` - The Scala version to compile with
- `platform.mainClass` - The main class to run

## Step 3: Create the Source Directory

Bleep uses a conventional directory structure:

```bash
mkdir -p myapp/src/scala/myapp
```

## Step 4: Write Your Code

Create `myapp/src/scala/myapp/Main.scala`:

```scala
package myapp

@main def main(): Unit =
  println("Hello from bleep!")
```

Your project structure should now look like:

```
my-first-bleep-project/
├── bleep.yaml
└── myapp/
    └── src/
        └── scala/
            └── myapp/
                └── Main.scala
```

## Step 5: Compile

Run the compile command:

```bash
bleep compile
```

The first run will:
1. Download the specified JVM (if needed)
2. Download Scala compiler and standard library
3. Start the Bloop build server
4. Compile your code

Subsequent compiles will be much faster since dependencies are cached.

## Step 6: Run

Run your application:

```bash
bleep run myapp
```

You should see:

```
Hello from bleep!
```

## Adding Dependencies

Let's add a dependency. Update your `bleep.yaml`:

```yaml
$schema: https://raw.githubusercontent.com/oyvindberg/bleep/master/schema.json
$version: 0.0.12

jvm:
  name: graalvm-community:21

projects:
  myapp:
    dependencies:
      - com.lihaoyi::fansi:0.5.0
    scala:
      version: 3.3.3
    platform:
      mainClass: myapp.Main
```

Note the `::` (double colon) for Scala dependencies. Java dependencies use a single `:`.

Update your `Main.scala` to use the library:

```scala
package myapp

import fansi._

@main def main(): Unit =
  val message = Color.Green("Hello") ++ Str(" from ") ++ Bold.On("bleep!")
  println(message)
```

Compile and run again:

```bash
bleep compile
bleep run myapp
```

You'll see colorized output in your terminal.

## Adding Tests

Let's add a test project. Update `bleep.yaml`:

```yaml
$schema: https://raw.githubusercontent.com/oyvindberg/bleep/master/schema.json
$version: 0.0.12

jvm:
  name: graalvm-community:21

projects:
  myapp:
    dependencies:
      - com.lihaoyi::fansi:0.5.0
    scala:
      version: 3.3.3
    platform:
      mainClass: myapp.Main

  myapp-test:
    dependsOn: myapp
    dependencies:
      - org.scalameta::munit:1.0.0
    scala:
      version: 3.3.3
    isTestProject: true
```

Create the test directory and file:

```bash
mkdir -p myapp-test/src/scala/myapp
```

Create `myapp-test/src/scala/myapp/MainTest.scala`:

```scala
package myapp

class MainTest extends munit.FunSuite:
  test("example test"):
    assertEquals(2 + 2, 4)
```

Run the tests:

```bash
bleep test myapp-test
```

## IDE Setup

To import your project into an IDE:

### IntelliJ IDEA

1. Run `bleep setup-ide`
2. Open IntelliJ and select "Open" (not "Import")
3. Navigate to your project directory
4. IntelliJ will detect BSP and import instantly

### VS Code with Metals

1. Run `bleep setup-ide`
2. Open the project folder in VS Code
3. Metals will detect the BSP configuration and import

## Project Structure Reference

Here's the complete project structure after this tutorial:

```
my-first-bleep-project/
├── bleep.yaml
├── myapp/
│   └── src/
│       └── scala/
│           └── myapp/
│               └── Main.scala
└── myapp-test/
    └── src/
        └── scala/
            └── myapp/
                └── MainTest.scala
```

Each project in `bleep.yaml` corresponds to a directory with `src/scala/` (or `src/java/`) underneath.
