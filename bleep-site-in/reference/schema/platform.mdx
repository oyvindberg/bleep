# Platform Configuration

Platform configuration specifies JVM, JavaScript, or Native targets.

## JVM Platform

```yaml
projects:
  myapp:
    platform:
      name: jvm
      mainClass: com.example.Main
      jvmOptions:
        - -Xmx2g
        - -XX:+UseG1GC
```

## Scala.js Platform

```yaml
projects:
  myapp:
    platform:
      name: js
      jsVersion: 1.14.0
      jsMode: release  # or debug
      jsKind: application  # or library
      mainClass: com.example.Main
```

## Scala Native Platform

```yaml
projects:
  myapp:
    platform:
      name: native
      nativeVersion: 0.4.16
      nativeMode: release  # or debug
      mainClass: com.example.Main
```

## Platform Fields

### JVM

| Field | Description |
|-------|-------------|
| `mainClass` | Main class for `bleep run` |
| `jvmOptions` | JVM arguments |

### Scala.js

| Field | Description |
|-------|-------------|
| `jsVersion` | Scala.js version |
| `jsMode` | `debug` or `release` |
| `jsKind` | `application` or `library` |
| `mainClass` | Entry point |

### Scala Native

| Field | Description |
|-------|-------------|
| `nativeVersion` | Scala Native version |
| `nativeMode` | `debug` or `release` |
| `mainClass` | Entry point |
