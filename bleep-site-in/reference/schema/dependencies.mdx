# Dependencies Schema

Dependencies are external libraries from Maven repositories.

## Basic Syntax

### Scala Dependencies

```yaml
dependencies:
  - com.lihaoyi::fansi:0.5.0
```

Uses `::` to append Scala version automatically.

### Java Dependencies

```yaml
dependencies:
  - org.slf4j:slf4j-api:2.0.9
```

Uses `:` for Java (no Scala suffix).

## Full Syntax

```yaml
dependencies:
  - module: com.example::mylib:1.0.0
    configuration: provided
    classifier: sources
    exclusions:
      - org.unwanted:*
    for3Use213: true
    publication:
      name: custom-name
      type: jar
```

## Fields

| Field | Description |
|-------|-------------|
| `module` | Maven coordinates |
| `configuration` | Scope (compile, test, provided) |
| `classifier` | Artifact classifier |
| `exclusions` | Transitive dependencies to exclude |
| `for3Use213` | Use Scala 2.13 artifact for Scala 3 |

## Version Syntax

```yaml
# Exact version
- com.lihaoyi::fansi:0.5.0

# Version range
- com.lihaoyi::fansi:[0.4.0,0.6.0)

# Latest
- com.lihaoyi::fansi:latest.release
```

Fixed versions are recommended for reproducibility.
