# bleep import

Import an existing sbt build.

## Synopsis

```bash
bleep import
```

## Description

Converts an sbt build into `bleep.yaml` by reading the Bloop JSON files that sbt generates.

## Prerequisites

Generate Bloop files from your sbt project first:

```bash
sbt bloopInstall
```

## Usage

```bash
# In your sbt project directory
sbt bloopInstall
bleep import
```

## What Gets Imported

- Projects and their structure
- Dependencies (Maven coordinates)
- Scala versions
- Source directories
- Compiler options
- Test frameworks

## What Needs Manual Work

- sbt plugins → Convert to [scripts](/docs/tutorials/your-first-script)
- Custom tasks → Convert to scripts
- Resource generators → Convert to source generators
