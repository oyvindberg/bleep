# `bleep build`

<p>create, edit, inspect, and refactor bleep.yaml</p>

## Synopsis

```bash
bleep build <subcommand> [args] [flags]
```

## Subcommands

- [`bleep build create-directories`](./create-directories): create all source and resource folders for project(s)
- [`bleep build normalize`](./normalize): normalize build (deduplicate, sort, etc)
- [`bleep build templates-reapply`](./templates-reapply): re-merge templates into projects (run after editing a template definition to propagate changes)
- [`bleep build project-rename`](./project-rename): rename a project; references in dependsOn and templates update automatically
- [`bleep build project-merge-into`](./project-merge-into): merge the first project's sources, dependencies, and config into the second; the first project is deleted
- [`bleep build projects-move`](./projects-move): move one or more projects into a different parent folder on disk
- [`bleep build templates-generate-new`](./templates-generate-new): throw away existing templates and infer new
- [`bleep build update-deps`](./update-deps): update every dependency in bleep.yaml to its newest available version
- [`bleep build update-dep`](./update-dep): update a single dependency (or every dependency from one organization) to the newest available version
- [`bleep build move-files-into-bleep-layout`](./move-files-into-bleep-layout): move source files around from sbt file layout to bleep layout. Your build will no longer have any `sbt-scope` or `folder` set.
- [`bleep build diff`](./diff): diff the effective (template-resolved) project config between git revisions
- [`bleep build show`](./show): show project configuration as YAML, either as written or with templates applied
- [`bleep build evicted`](./evicted): report dependency conflicts (evictions) so you can decide which library version schemes apply
- [`bleep build invalidated`](./invalidated): list projects whose source or config changed vs a base commit (and their transitive dependents)
- [`bleep build new`](./new): scaffold a new bleep workspace in the current directory: bleep.yaml, a project, a sibling test project, hello-world source
